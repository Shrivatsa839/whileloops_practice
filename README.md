# Python While Loops Practice

This repository contains beginner-friendly Python programs that demonstrate the use of **while loops**. These exercises were completed to strengthen my understanding of looping concepts and basic problem-solving in Python.

## 📚 Programs Included

1. Print numbers from 1 to 10
2. Print numbers from 10 to 1
3. Print all even numbers between 1 and 100
4. Print all odd numbers between 1 and 100
5. Print the multiplication table of a given number
6. Calculate the sum of the first `n` natural numbers
7. Calculate the sum of all even numbers from 1 to `n`
8. Calculate the sum of all odd numbers from 1 to `n`
9. Calculate the factorial of a given number
10. Find the product of all digits of a given number

## 🛠️ Technologies Used

- Python 3

## 🎯 Purpose

This repository is part of my Python learning journey. The goal is to practice programming fundamentals, improve logical thinking, and build a strong foundation in Python.

   ```bash
   git clone https://github.com/Shrivatsa839/whileloops_practice.git
   ```

